# Orchestrator Blueprint

## Purpose
Coordinate multiple sub-agents or processes using the L0 kernel as foundation.

## Components
- Sub-agent lifecycle manager
- Workflow definition interpreter
- Result aggregation and routing

## Boundaries
- Sub-agents must also respect L0 governance
- No privilege escalation through orchestration

## Integration
- Uses L1 controller for evolution workflows
- Defines workflow templates in L1/workflows/
