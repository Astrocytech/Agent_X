# FIC-L1-006: FIC Validator

**fic_id:** `FIC-L1-006`
**unit_id:** `UNIT-L1-006`
**version:** `v0.6.0`
**status:** `ready-for-code`
**target_file:** `L1/controller/fic_validator.py`

## Description

Validates `FicTemplate` records generated by the FIC Generator. Checks structural integrity: required fields, naming conventions, uniqueness, and plausible values. Returns a `FicValidationResult` with pass/fail, errors, and warnings.

## Public surface

```python
__all__ = [
    "FicValidationResult",
    "FicValidator",
    "FicValidatorError",
    "validate_fics",
]
```

### Exports

- `FicValidationResult` — frozen dataclass: `is_valid`, `errors`, `warnings`
- `FicValidatorError` — base exception
- `FicValidator` — class:
  - `validate(self, templates: tuple[FicTemplate, ...]) -> FicValidationResult`
- `validate_fics(templates) -> FicValidationResult`

## Validation rules

| Check | Error condition |
|---|---|
| Type check | Input must be a tuple of FicTemplate |
| Empty check | Empty tuple → error |
| fic_id pattern | Must match `FIC-L1-PLAN-\d{3}` |
| unit_id pattern | Must match `UNIT-L1-PLAN-\d{3}` |
| target_file | Must be non-empty string; must not be absolute |
| description | Must be non-empty |
| status | Must be `"draft"` (only accepted value for generated templates) |
| version | Must be `"v0.1.0"` (only accepted value for generated templates) |
| fic_id uniqueness | No duplicate fic_ids |
| unit_id uniqueness | No duplicate unit_ids |

## Dependency contract

- **stdlib only** (dataclasses, re, typing)
- May import `FicTemplate` from `L1.controller.fic_generator`
- **No** `os`, `subprocess`, `requests`, `urllib`, `socket`, `http`
- **No** imports from `L0` or `L2`

## Rules

1. `validate` never raises — always returns a `FicValidationResult`.
2. All errors are collected (not short-circuited on first failure).
3. `is_valid` is `True` only when `errors` is empty.
4. `warnings` is always a tuple (may be empty).

## Edge cases

| Case | Behavior |
|---|---|
| empty tuple | is_valid=False, error="no templates" |
| None | is_valid=False, error="must be a tuple" |
| wrong type elements | is_valid=False, error="must be FicTemplate" |
| single valid template | is_valid=True |
| many valid templates | is_valid=True |
| one invalid among many | is_valid=False, error for specific template |
| duplicate fic_ids | is_valid=False, error="duplicate fic_id" |
