# Schemas

agentx-init uses JSON Schema (draft-07) for all structured data.

## Schema Files

| File | Purpose |
| --- | --- |
| `schemas/config.schema.json` | Configuration file schema |
| `schemas/repo_scan.schema.json` | Repository scan output schema |
| `schemas/architecture_report.schema.json` | Architecture report schema |
| `schemas/governance_decision.schema.json` | Governance decision schema |
| `schemas/evolution_plan.schema.json` | Evolution plan schema |
| `schemas/patch_proposal.schema.json` | Patch proposal schema |
| `schemas/validation_report.schema.json` | Validation report schema |
| `schemas/audit_event.schema.json` | Audit event schema |
| `schemas/graph_node.schema.json` | Graph node schema |
| `schemas/graph_edge.schema.json` | Graph edge schema |
| `schemas/graph_snapshot.schema.json` | Graph snapshot schema |
| `schemas/graph_query_result.schema.json` | Graph query result schema |
| `schemas/graph_manifest.schema.json` | Graph manifest schema |
| `schemas/graph_integrity.schema.json` | Graph integrity schema |
| `schemas/completion_record.schema.json` | Common implementation-completion record schema |
| `schemas/path_registry.schema.json` | Path registry schema |
| `schemas/runtime_paths.schema.json` | Runtime path configuration schema |
| `schemas/config_validation_report.schema.json` | Config validation report schema |

All schemas are located in `agentx_initiator/schemas/`.
