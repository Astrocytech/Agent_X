# Non-Goals

agentx-init is not:

- an L0 component
- a self-modifying runtime
- an autonomous coding agent
- a web UI
- an API server
- an autonomous patch applicator
- an auto-promotion engine
- a multi-agent orchestrator
- a swarm behavior system
- a remote GitHub automation tool
- a daemon or scheduler
- a runtime self-modification system

These capabilities are intentionally excluded from version 1.
